CREDIT CARD ITEM CONFIGURATION
================================

To set the display name of the Credit_Card item to "Credit Card", you need to add it to your item configuration file (typically CfgMaster.sqf or similar).

LOCATION: CRP_Client.cmr_cicada\configuration\CfgMaster.sqf

ADD THIS ENTRY:
---------------
INV_Item("Credit_Card", "Credit Card", "data\images\items\creditcard.paa", "scripts\items\creditcard.sqf", "Credit Card", [["money", 0]]);

OR if your format is different, look for other item definitions in CfgMaster.sqf and match the format, ensuring the second parameter (display name) is "Credit Card".

The item class name is "Credit_Card" and the display name should be "Credit Card".
